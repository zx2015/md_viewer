# Wikilinks

Obsidian/Logseq-style internal links.

- [[basic]] — link to another file
- [[gfm]] — link by name
- [[code|the code sample]] — link with alias
- [[does-not-exist]] — dead link (shown as plain text, becomes styled as `.wikilink.dead` in CSS)

Wikilinks work both in paragraphs and lists.
